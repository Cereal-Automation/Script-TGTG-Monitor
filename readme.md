# TGTG Monitor Script

The TGTG Monitor is a powerful automation script that continuously monitors Too Good To Go (TGTG) for available Magic
Bags in your area. Never miss out on great food deals again â€“ get instant notifications when items become available at
your favorite stores!

## **Features**

- **Real-time Monitoring**: Continuously checks for new Magic Bags and stock availability in your specified area
- **Location-based Search**: Configure your exact location using latitude/longitude coordinates and set a custom search
  radius
- **Dual Monitoring Strategies**:
    - **New Item Detection**: Get notified when completely new stores or items appear in your area
    - **Stock Availability**: Receive alerts when existing stores restock their Magic Bags
- **Secure Authentication**: Simple email-based login with polling mechanism for secure TGTG account access
- **Proxy Support**: Optional proxy configuration with automatic rotation for enhanced privacy and reliability
- **Customizable Intervals**: Set your preferred monitoring frequency to balance between responsiveness and resource
  usage
